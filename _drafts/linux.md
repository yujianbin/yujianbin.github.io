切换到root用户
```
su
```
默认是切换到root用户，需要输入密码，但是root密码是随机的，我们不知道。所以需要先改root密码；

```
sudo passwd
```
输入当前用户密码；
输入root用户密码；

然后在使用 `su` 切换到root用户。


当你用vi打开一个文件后，因为文件太长，如何才能找到你所要查找的关键字呢？
在vi里可没有菜单-〉查找

不过没关系，你在命令模式下敲斜杆( / )这时在状态栏（也就是屏幕左下脚）就出现了 “/” 然后输入你要查找的关键字敲回车就可以了。
如果你要继续查找此关键字，敲字符 n 就可以继续查找了。
敲字符N（大写N）就会向前查询；
